7. Flexbox

Flexbox brukes til å justere elementer horisontalt og vertikalt. Det gir mer fleksible layouter.

Forklaring av egenskapene:

  • justify-content: space-between; - Plasserer elementene med jevn avstand mellom seg.
  • align-items: center; - Justerer elementene vertikalt i midten av containeren.
  • gap: 10px; - Gir en mellomrom på 10px mellom elementene.

8. Animasjoner i CSS

Med CSS kan vi lage enkle eller komplekse animasjoner.

Eksempel: Bakgrunnsfargen endres


        @keyframes changeColor {
            0% { background-color: red; }
            50% { background-color: blue; }
            100% { background-color: red; }
        }
        
        .animated-box {
            width: 100px;
            height: 100px;
            background-color: red;
            animation: changeColor 3s infinite;
        }

Resultat:

Eksempel: Flytte en boks fra venstre til høyre


@keyframes slideRight {
    0% { transform: translateX(0); }
    100% { transform: translateX(200px); }
}

.moving-box {
    width: 100px;
    height: 100px;
    background-color: lightblue;
    animation: slideRight 2s infinite alternate;
}

Resultat:

Eksempel: En pulserende sirkel


@keyframes pulse {
    0% { transform: scale(1); opacity: 1; }
    50% { transform: scale(1.2); opacity: 0.7; }
    100% { transform: scale(1); opacity: 1; }
}

.pulse-circle {
    width: 50px;
    height: 50px;
    background-color: orange;
    border-radius: 50%;
    animation: pulse 1.5s infinite;
}

Resultat:

Eksempel: En knapp som vokser ved hover


.button-hover {
    padding: 10px 20px;
    background-color: green;
    color: white;
    border: none;
    cursor: pointer;
    transition: transform 0.3s ease-in-out;
}

.button-hover:hover {
    transform: scale(1.2);
}

9. Responsivt Design

Media queries gjør at nettsider tilpasser seg ulike skjermer.

@media (max-width: 600px) {
        body {
            background-color: lightgray;
        }
    }

Resultat:

Endrer bakgrunnsfarge på små skjermer.

Eksempel: Responsivt design


/* Standard stil for store skjermer */
body {
    font-size: 18px;
    background-color: white;
}

/* Mobilvisning (maks 480px bredde) */
@media (max-width: 480px) {
    body {
        font-size: 16px;
        background-color: lightgray;
    }
}

/* Eksempel på flere media queries */

/* 1. Mobilvisning (maks 480px) */
@media (max-width: 480px) {
    body {
        background-color: #f4f4f4;
        font-size: 16px;
    }
    nav ul {
        flex-direction: column;
    }
}

/* 2. Nettbrett (481px - 768px) */
@media (min-width: 481px) and (max-width: 768px) {
    body {
        background-color: #e0e0e0;
    }
}

/* 3. Laptop & PC (768px - 1440px) */
@media (min-width: 769px) and (max-width: 1440px) {
    body {
        background-color: white;
    }
}

/* 4. Store skjermer (1441px +) */
@media (min-width: 1441px) {
    body {
        background-color: lightblue;
    }

Resultat:

Åpne denne nettsiden på en liten skjerm eller bruk utviklerverktøyet i nettleseren (F12 -> Toggle device toolbar) for å se endringen.

15. CSS Grid vs. Flexbox

Flexbox er best for én-dimensjonale layout (rekke eller kolonne), mens CSS Grid er best for komplekse to-dimensjonale layout.

Eksempel: Grid med tre kolonner


        .grid-container {
            display: grid;
            grid-template-columns: repeat(3, 1fr);
            gap: 10px;
        }
        
        .grid-item {
            background-color: lightgray;
            padding: 20px;
            text-align: center;
        }
