Korleis lage ei nettside?

Ting som må på plass før ein startar

  • Last ned og installer Visual Studio Code

  • Last ned Live Server extension i Visual Studio Code (VSCode)

  • Opprett ei mappe på PCen din, kor du samlar all kode. (Kvart prosjekt / nettside skal ha si eiga mappe.)

Opprette HTML dokument

w3schools

Ei HTML side inneheld:

  • <!doctype html> :

  • <html> <html/> :

  • <head> <head/> : Det som ikkje er ein del av sjølve nettsida (alt det me ikkje kan sjå)

  • <meta charset="utf-8">: Henter inn tegnsettet UTF-8 (slik at ein får alle tegna)

  • <title> <title/> : Tittel på sida (det som står i fana i nettlesaren)

  • <link rel="stylesheet"> <link/> :

For å opprette ei HTML side i visual studio code kan ein gjere slik:

  1. Klikk File --> New Text File
  2. Klikk "Select language" og skriv inn html
  3. Lagre fila ved å klikke Ctrl + s (husk å lagre fila di i rett mappe)

Filendelsen til ei html fil er .html

Obs! HTML filer skrives med berre små bokstavar. Unngå lange navn og mellomrom.

Eksempel: index.html, about.html, homepage.html

Bilete

w3schools

Bilete i HTML bruker taggen <img>

Eksempel:

<img src="images/cat.png" alt="Katt"/>

src viser til lenken/filbanen til bilete. Denne må bruke relativ url (ikkje C:\ osv)

alt er alternativ tekst. Denne blir lest opp dersom ein brukar opplesning av nettsider. Det er dette som visar dersom bilete ikkje kan laste inn.

Tabellar

w3schools

Ein tabell i html skrives inn ved å bruke <table> taggen

<table> er ikkje så mykje i bruk lenger, ein bruker for det meste <div> taggen for å opprette noko som liknar tabellar

For å opprette ein tabell, gjer ein slik:

<table>

 <tr>

  <th>Fornavn</th>

  <th>

  </th>

 </tr>

 <tr>

  <td>

  </td>

  <td>

  </td>

 </tr>

</table>


Der <table> er starten på tabellen

<tr> er ei rad i tabellen (table row)

<th> er overskriften (table header)

<td> er dataen som ligg i cella (table data)
