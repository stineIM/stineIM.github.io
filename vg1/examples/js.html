variables.js

Ein variabel er eit navn som representerer en verdi.

Variabel med tal (number) som datatype:

let number = 1;

Variabel med tekst (string) som datatype:

let firstName = "Stephen";

Aritmetiske operatorar (matematikk)

Dei fire rekneartane som me kjenner frå matematikken, kallar me for aritmetiske operatorar.

Operatorane som tilsvarer dei fire rekneartane, er + (pluss), - (minus), * (gange) og / (dele)

Det er god praksis å bruke mellomrom på kvar side av operatoren, det gjer koden lett å lese.

Arrays

w3schools

Arrays er "lister" som inneheld mange verdier.

Ein oppretter ein array ved å lage ein variabel og sette den til å vere []

Eksempel:

let temperatures = [10,12,15,12,14,13,9]

På denne måten kan ein lagre fleire verdiar i ein variabel, slik at ein ikkje treng å opprette ein variabel for kvar verdi i arrayen.

Eitt array kan innehalde mange datatypar, tal, tekst, objekt osv.

Array metoder

Arrays inneheld ein del metodar som ein kan bruke for å gjer endringar i arrayen.

APIs

API står for Application Programming Interface.

A Web API is an application programming interface for the Web.

A Browser API can extend the functionality of a web browser.

A Server API can extend the functionality of a web server.

Eit API gjer data tilgjengeleg for andre.

Det finnes mange tredjepartsløsninger som har eit API tilgjengeleg.

Eksempel:

  • Meta API
  • Twitter API
  • YouTube API
  • Vær API
  • Oslo bysykkel
